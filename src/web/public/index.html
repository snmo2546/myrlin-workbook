How agent workflows work

Single agent -- One model handles everything. Full conversation context preserved. Best for complex tasks where the agent needs to remember all decisions.

Model per stage -- Use Haiku for planning (fast/cheap exploration), Sonnet for running (good balance), Opus for review (thorough analysis). Configure defaults in Settings > Advanced.

Agent teams -- Multiple agents work in parallel on sub-tasks. Each gets focused context. Enable the flag above to let Claude spawn subagents. Monitor progress in the session detail panel.
